C Programming Example for Mobile/Tab

This is a simple C program example for mobile/tablet development (e.g., using libraries like SDL for cross-platform apps).

#include <stdio.h>

int main() {
    printf("Hello, Mobile/Tab World!\n");
    printf("This is a basic C program for mobile/tablet.\n");
    return 0;
}

Compile with: gcc -o mobile_app mobile.c (assuming SDL or similar for mobile integration).
